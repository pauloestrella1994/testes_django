# Testes no Django
Projeto de uma sorveteria com testes automatizados usando o Django

### Para rodar localmente

Para criar o arquivo .env:
```
$ make env
```
Para criar as migrações no db.sqlite3:
```
$ make makemigrations
```
Para migrar: 
```
$ make migrate
```
Para rodar os testes automatizados:
```
$ make test
```
